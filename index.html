Features

Why should you purchase one?

Four Opcodes

LOAD: load constant into accumulator
ADD: add constant to accumulator
LSL: left shift left value in accumulator
JMP: branch to address

Push Button Master Clock

To be pressed by the user, completely de-bounced circuit. Using both falling and rising edge.

Programmable Read-Only Memory

4 data words, 6 bits each. Words made up from a 2 bit opcode and 4 bits of data.

Visible Inner Workings

Ideal for students to examine the components. Convenient learning tool.
