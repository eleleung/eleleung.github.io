About

Early Dementia Detection

Ongoing Monitoring

The condition of dementia patients may be frequently assessed from home using the app.

Doctors

Doctors can create and prescibe specialised tests for their patients using building blocks provided.

Carers

Carers can administer tests from the comfort of home and upload the results to doctors.

Patients

Patients can recieve more frequent and refined updates on the state of their condition.

Features

Early Dementia Detection includes the following features

Mobile and Web

Accessible on both mobile and web platforms.

iOS and Android

Supported on both iOS and Android.

Custom Test Creator

Provides building blocks for creating specialised tests.

Test Sharing

Enables tests to be shared by doctors.

Text-to-Speech

Text-to-Speech functionality for transcribing patient speech.

Photo Uploading

Enables photos to be taken and uploaded to doctors.

Questionnaires

Provides questionnaires for assessing patients.
